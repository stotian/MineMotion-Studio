# Phase 16.8 - Movement And Trail VFX

Ten native movement recipes extend the shared built-in registry: dash, weapon,
projectile, footsteps, running, falling, flying, Elytra, Ender pearl, and
swimming. The first seven satisfy the required Phase 16 family; the three
Minecraft-native variants bring the catalog to 60 native recipes and 72 total
entries without claiming stability before preview and regression gates exist.

Every exposed parameter has an automated output-influence regression. All
recipes use the existing schema 10 effect collection, deterministic primitive
evaluator, global frame budget, Three.js preview, and export paths. Their
combined final-quality regression allocates 266 particles and 1,456 segments.

The oversized legacy repair path now validates synchronized native records
directly instead of cloning and rebuilding them. Repairable invalid legacy
parameters remain editable, corrupt native data still fails closed, and the
4,097-effect characterization runs in 0.95 seconds in focused measurement.
