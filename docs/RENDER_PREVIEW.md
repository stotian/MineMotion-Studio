# Render Preview

Render Preview shows the scene as a cinematic preview mode using the current
project settings.

## Includes

- active camera label
- post-processing overlays
- cinematic bars
- screen effects
- world effects
- timeline playback

## Render Settings

- resolution preset: 720p, 1080p, 1440p, 4K, custom
- aspect ratio: 16:9, 2.35:1, 1:1, 9:16
- render preview enabled flag
- cinematic bars enabled flag
- cinematic bars ratio

## Current Limits

Phase 3 can export PNG stills and PNG sequence ZIP files from the viewport.
WebM recording is available when the browser supports `MediaRecorder`; MP4
requires a future native/FFmpeg pipeline.
